# ollama_web
Ollama_web is a python project to allow the user to communicate with his Ollama AI model hosted in his server via a web interface

## Usage
Install the program on your machine with the necessary dependency on.

## Error Code
- 401: The program can't pull any model from the device
- 402: The program can't find the choosen model

## Dependency
- Python 3.12
- python-ollama
- python-flask
